# Sistema Acadêmico - Hackathon KizaDev

Este projeto foi desenvolvido durante o Hackathon da KizaDev com o objetivo de criar um sistema acadêmico com autenticação e gerenciamento de alunos.

O sistema possui funcionalidades de:

- cadastro de usuários
- login
- cadastro de alunos
- listagem de alunos
- edição de alunos
- exclusão de alunos

A proposta foi construir uma aplicação funcional, simples e organizada, simulando um sistema acadêmico básico.

## Tecnologias utilizadas

- HTML
- CSS
- JavaScript
- Java
- Spring Boot
- PostgreSQL
- Render

## Como executar o projeto

Para avaliação, não é necessário instalar dependências ou configurar ambiente local, pois o sistema já está disponível online.

Acesse:

https://projeto-do-hackton-login-1.onrender.com

### Como utilizar

- Para acessar o sistema, é necessário primeiro realizar o cadastro de usuário
- Após o cadastro, basta fazer o login normalmente com as credenciais criadas (e-mail e senha)

Observação: o primeiro acesso pode demorar alguns segundos devido ao cold start do Render, que ocorre quando a aplicação fica um período sem uso.

Caso seja necessário rodar localmente, o projeto backend utiliza Java com Spring Boot e pode ser executado via IDE (Eclipse/IntelliJ) ou com Maven.

## Observações

Este projeto foi desenvolvido em contexto de Hackathon, com foco em entrega funcional dentro do prazo. Por isso, algumas melhorias futuras ainda podem ser feitas em pontos como organização interna, validações e refinamentos técnicos.
